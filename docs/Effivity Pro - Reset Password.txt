EFFIVITY

Reset Password

Abstract

This document guides the users to reset password in Effivity.

Effivity Tech Support
techsupport@effivity.com

Open Effivity domain page using the below URL.

https://www.effivity.com/WebApp/account/domain

Enter the respective domain.

Choose the forgot password option.

Enter respective mail id and submit.

Password reset link will be sent to the entered mail.

The user can use the link sent through email to reset their password.

We hope this document was helpful.
